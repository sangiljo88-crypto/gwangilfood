저자: 정지원, 유지은, 원충열
요약 목적: 랜딩페이지/상세페이지 사업 적용

====================================
[ 핵심 개념: 맥락(Context)이란? ]
====================================

▶ 맥락의 정의
- "사물 따위가 서로 이어져 있는 관계나 연관"
- Context = Con(함께) + Texture(직물을 짜다)
- 직물을 짜나가듯 무언가를 함께 엮어서 만들어내는 것

▶ 왜 지금 맥락인가?
- 스토리(Story): 왕이 죽고 왕비도 죽었다 (시간순 나열)
- 플롯(Plot): 왕이 죽자 슬픔을 못 이겨 왕비도 죽었다 (인과관계)
→ 브랜드도 각 요소의 '유기적 연결'과 '인과관계'가 중요!

▶ 브랜드 성패의 핵심
- 브랜드의 고유한 맥락이 성패를 좌우
- 콘셉트, 스토리텔링 모두 '맥락'의 구성 요소이거나 맥락에서 뻗어나가는 가지
- 맥락은 쉽게 무너지지 않는 브랜드의 존재 이유

====================================
[ 1. '무엇을'보다 '왜', '어떻게'가 중요한 시대 ]
====================================

▶ 굿즈 열풍의 진실
- 과거: 책을 사면 사은품이 딸려옴
- 현재: 굿즈를 사면 책이 '덤'으로 딸려옴
- 핵심: "책을 사는 새로운 맥락이 생겨났다"

★ 랜딩페이지 적용 포인트
→ 고객이 제품을 사는 '새로운 이유'를 발견하라
→ 단순히 제품 스펙을 나열하지 말고, 고객의 구매 맥락을 이해하라

▶ 새로운 소비 맥락 발견하기
- 알라딘의 사례: 굿즈 때문에 책을 사는 소비자 12% (2016년)
- 마케터는 새로운 맥락을 먼저 '발견'하고 '창조'해야 함
- 대부분의 맥락은 나중에 발견되므로, 고객 행동을 예민하게 관찰해야 함

====================================
[ 2. 고객의 맥락과 브랜드의 맥락을 만나게 하라 ]
====================================

▶ 맥락의 3가지 교집합
1. 소비의 맥락: 고객이 왜 이것을 사는가?
2. 기업의 맥락: 우리는 왜 이것을 파는가?
3. 시대의 맥락: 지금 왜 이것이 필요한가?

★ 랜딩페이지 적용 포인트
→ 이 3가지가 만나는 지점에서 의미가 탄생
→ 상세페이지는 이 교집합을 시각화하는 작업

▶ 성공 사례: 아마존, 테슬라, 샤오미
- 아마존: "고객 우선"이라는 맥락 위에 모든 사업 전개
- 테슬라: "지속가능한 에너지"라는 맥락으로 사회적 동의 획득
- 샤오미: "고객을 친구로"라는 참여형 소비를 맥락으로 이념화

====================================
[ 3. 브랜드의 오리진에 맥락의 단서가 있다 ]
====================================

▶ 자기다움의 출발점
- 브랜드가 왜 만들어졌는지
- 창업자의 철학은 무엇인지
- 해결하려는 문제는 무엇인지

★ 랜딩페이지 적용 포인트
→ "왜 이 제품을 만들었는가?"를 스토리로 전달
→ 창업 스토리, 제품 개발 배경을 맥락으로 연결
→ 단순 제품 소개가 아닌 '존재 이유' 제시

▶ 다름은 표현하고, 나음은 증명하라
- 다름(Difference): 타사와 무엇이 다른가? → 표현만으로 충분
- 나음(Better): 타사보다 더 좋은가? → 반드시 증명 필요

예시)
- "우리는 친환경 소재를 사용합니다" (다름) → 표현
- "우리 제품이 더 오래 갑니다" (나음) → 테스트 결과, 수치, 인증 필요

====================================
[ 4. 삶을 해독하고 브랜드를 코딩하라 ]
====================================

▶ 미디어 오디언스 시대
- 과거: 미디어는 일방적 송출 도구
- 현재: 소비자가 미디어이자 오디언스(참여자)
- SNS, 유튜브 → 소비자가 직접 콘텐츠 생산

★ 랜딩페이지 적용 포인트
→ 공유하고 싶게 만드는 콘텐츠
→ 인스타그래머블(Instagrammable) 요소 삽입
→ 고객 리뷰, 사용 후기를 전면 배치

▶ Z세대의 특징
- 태어날 때부터 디지털 원주민
- 진정성을 중요시
- 완벽한 척보다 솔직함을 선호
- 참여와 공유가 일상

====================================
[ 5. 브랜드 경험을 기획하고 실행하라 ]
====================================

▶ 접점(Touch Point) 설계
- 고객이 브랜드를 경험하는 모든 순간
- 온라인/오프라인 모든 접점이 일관된 맥락 유지

★ 랜딩페이지는 핵심 접점!
→ 첫 인상을 결정하는 가장 중요한 순간
→ 브랜드의 맥락이 명확하게 전달되어야 함

▶ 온라인 시대의 오프라인
- 온라인이 발달할수록 오프라인의 가치 상승
- 오프라인은 '경험'을 제공하는 공간
- 랜딩페이지는 온라인 공간이지만 '경험'을 설계해야 함

예시)
- 츠타야 서점: 단순 서점이 아닌 라이프스타일 제안
- 무인양품 헛: 실제 생활 체험 공간
→ 랜딩페이지도 '체험'을 시뮬레이션할 수 있어야 함

====================================
[ 6. 맥락은 확산을 위한 설계다 ]
====================================

▶ 도달보다 확산의 미디어가 중요
- 과거: 얼마나 많은 사람에게 도달했는가?
- 현재: 얼마나 많이 공유되고 확산되는가?

★ 랜딩페이지 적용 포인트
→ 공유 버튼 배치
→ 공유하고 싶은 명언, 인포그래픽 삽입
→ "친구에게 추천하고 싶은" 요소 강조

▶ 공감의 코드 제공
- 고객의 감정과 연결되는 메시지
- "나도 그래!" 하는 공감 포인트
- 특정 세대, 집단의 언어 사용

예시)
- 배달의민족: "다이어트는 내일부터"
- 막내 페르소나로 20~30대와 공감대 형성

====================================
[ 7. 맥락을 완성하는 것은 브랜드의 매력이다 ]
====================================

▶ 매력의 3가지 요소

1) 엣지(Edge)가 살아있는 매력
- 날카로운 모서리처럼 두드러지는 강점
- 속성 경쟁이 아닌 '차별점' 강조

예시: 삼성 세리프TV
- 기능 업그레이드가 아닌 '생활 오브제'로 포지셔닝
- 가구 디자이너와 협업
- 백화점, 고급 가구매장에서만 판매
- TV 카테고리가 아닌 독립 카테고리

★ 랜딩페이지 적용
→ "우리만의 특별함"을 명확히
→ 경쟁사와 다른 점을 한 문장으로

2) 공감력 있는 매력
- 고객의 관심사 = 브랜드의 관심사

예시: 구글 두들(Google Doodle)
- 세월호 추모, 아인슈타인 생일, 브라질 월드컵 등
- 인류의 공통 관심사와 함께하는 공감의 힘

예시: 에어비앤비 가이드북
- "지금 가장 핫한 제주", "지금 가장 핫한 전주"
- 숙박보다 '여행 경험' 자체에 집중
- "여행은 살아보는 거야" 철학 실현

★ 랜딩페이지 적용
→ 제품 기능만 나열하지 말고
→ 고객의 삶, 관심사와 연결
→ "이 제품이 당신의 삶을 어떻게 바꾸는가?"

3) 유머 있는 매력
- 가장 고전적이지만 가장 근본적인 매력 요소
- 유머가 통하면 고객은 무장해제되고 관대해짐

예시: 배달의민족
- "우리가 어떤 민족입니까?"
- "다이어트는 내일부터"
- "이런 16기가" 굿즈
- 치믈리에(치킨감별사) 자격시험

예시: 앱솔루트 보드카
- 유니크한 병 모양을 활용한 예술적 광고
- 광고를 수집하는 팬덤 형성
- 촛불집회를 병 모양으로 형상화한 위트

★ 랜딩페이지 적용
→ 진지함과 유머의 균형
→ 타겟층의 유머 코드 이해
→ B급 감성도 전략이 될 수 있음

====================================
[ 8. 기술로 설계된 경험을 제안하라 ]
====================================

▶ 기술과 브랜드의 연관성 설계

예시: 볼보 라이프페인트(Life Paint)
- 자전거/옷에 뿌리는 투명 스프레이
- 햇빛에는 안 보이지만 밤에 헤드라이트에 반사
- "충돌에서 살아남는 가장 좋은 방법은 충돌을 피하는 것"
- 안전이라는 브랜드 본질을 기술로 구현

예시: 보이스미러(Voice Mirror)
- 시각장애인 화장 도움 앱
- 셀카 찍으면 뷰티 서포터즈가 피드백
- "아름다워지고 싶은 건 시력과 무관"
- 아름다움 추구라는 화장품 본질을 기술로 구현

★ 핵심 원칙
- 기술 그 자체가 아니라 '기술이 주는 의미'가 중요
- 최신 기술이 아니어도 됨
- 브랜드 본질과 연결되어야 함

[ 실전 적용: 랜딩페이지 제작 체크리스트 ]
====================================

[ 1단계: 맥락 발견 ]
□ 고객은 왜 이 제품이 필요한가? (소비의 맥락)
□ 우리는 왜 이 제품을 만들었는가? (기업의 맥락)
□ 지금 왜 이 제품이 중요한가? (시대의 맥락)
□ 위 3가지의 교집합은 무엇인가?

[ 2단계: 자기다움 정의 ]
□ 우리만의 오리진 스토리는?
□ 경쟁사와 '다른 점'은? (표현)
□ 경쟁사보다 '나은 점'은? (증명 자료 준비)
□ 우리의 엣지(Edge)는 무엇인가?

[ 3단계: 타겟 이해 ]
□ 우리 고객은 누구인가?
□ 그들의 언어는 무엇인가?
□ 그들이 공감하는 코드는?
□ 그들의 관심사는?

[ 4단계: 경험 설계 ]
□ 첫 화면에서 핵심 맥락이 보이는가?
□ 스크롤하며 '스토리'가 전개되는가?
□ 인과관계가 명확한가? (플롯)
□ 각 섹션이 유기적으로 연결되는가?

[ 5단계: 확산 설계 ]
□ 공유하고 싶은 요소가 있는가?
□ 인스타그래머블한 이미지/문구가 있는가?
□ 리뷰/후기가 전면에 배치되었는가?
□ SNS 공유 버튼이 적절히 배치되었는가?

[ 6단계: 매력 완성 ]
□ 공감 포인트가 명확한가?
□ 유머 요소가 적절히 들어갔는가?
□ 브랜드 개성이 느껴지는가?
□ "좋다"는 감정을 불러일으키는가?

[ 최종 핵심 메시지 ]
====================================

★ 맥락을 파는 5가지 원칙

1. 제품이 아니라 '이유'를 팔아라
   - 고객이 왜 사야 하는지
   - 우리가 왜 만들었는지
   - 지금 왜 필요한지

2. 스펙이 아니라 '스토리'를 팔아라
   - 시간순 나열(스토리)이 아닌
   - 인과관계(플롯)로 구성
   - 각 요소를 유기적으로 연결

3. 기능이 아니라 '경험'을 팔아라
   - 고객의 삶이 어떻게 바뀌는가?
   - 어떤 감정을 느끼게 하는가?
   - 어떤 가치를 제공하는가?

4. 완벽함이 아니라 '진정성'을 팔아라
   - 솔직하고 정직하게
   - 결점도 인정하며
   - 인간적인 매력으로

5. 도달이 아니라 '확산'을 설계하라
   - 공유하고 싶게
   - 참여하고 싶게
   - 이야기하고 싶게

====================================
[ 에필로그: 움직이는 과녁을 맞추는 법 ]
====================================

"움직이는 과녁을 어떻게 맞출 것인가?"

→ 과녁의 속도에 맞춰 함께 움직여라
→ 과녁을 맞추려 하지 말고, 과정 자체에 집중하라
→ 완결된 결말이 아닌, 지속적인 변화를 받아들여라

맥락은:
- 변화를 이해하는 관점
- 의미를 부여하는 도구
- 과정 그 자체

▶ 랜딩페이지 구조 예시

1. 히어로 섹션 (Hero Section)
   → 핵심 맥락을 한 문장으로
   → "왜"에 대한 답
   → 강렬한 비주얼

2. 문제 제기 (Problem)
   → 고객의 페인 포인트
   → 공감 코드
   → "나도 그래!" 반응 유도

3. 솔루션 제시 (Solution)
   → 우리 제품/서비스가 해결책
   → 다름과 나음 동시 제시
   → 스토리로 연결

4. 증거 제시 (Proof)
   → 나음의 증명
   → 수치, 데이터, 인증
   → 고객 후기, 리뷰

5. 경험 시뮬레이션 (Experience)
   → 사용 후 변화된 삶
   → Before & After
   → 감정적 혜택 강조

6. 행동 유도 (CTA)
   → 명확한 다음 단계
   → 확산 가능한 요소
   → 공유 독려
