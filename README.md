# 광일푸드 | 왕도매 멸치 프리미엄 랜딩페이지

## 프로젝트 개요
- **브랜드**: 광일푸드 (건어물 도소매 왕도매)
- **목적**: 영업용 샘플 랜딩페이지
- **대표 상품**: 코오푸드 국물멸치 육수용 다시멸치 식자재용 1.5kg (국내산)
- **타겟**: 45세 베테랑 주부 / 식당·반찬가게 운영자

## 완성된 기능
- ✅ PASONA 법칙 기반 전체 퍼널 구조
- ✅ 가격 앵커링 (마트 대비 왕도매 가격 비교)
- ✅ 상품 상세 + USP 3가지 (선별공정, 저온유통, 왕도매가격)
- ✅ 신뢰 구조 (왕도매 전문, HACCP 인증, 직접 소통)
- ✅ 고객 후기 3건 (G마켓 평점 4.3점 기반)
- ✅ FAQ 아코디언 5개
- ✅ 스마트스토어 연결 CTA (5곳 분산)
- ✅ 플로팅 구매 버튼 (스크롤 시 자연스럽게 등장)
- ✅ GSAP 스크롤 애니메이션 (폴백 포함)
- ✅ 섹션별 stagger 딜레이 (리듬감 있는 등장)
- ✅ 숫자 카운터 애니메이션 (1.5kg, 100건+, 4.3점, 70%↓)
- ✅ 완전 반응형 (모바일 퍼스트)
- ✅ 스크롤 프로그레스 바
- ✅ **로고 삽입 완료** (3곳: 헤더, 히어로 카드, 푸터)

## 이번 업데이트에서 개선된 사항

### 🎨 디자인 리파인
- 로고 컬러(골드 #D4AF37, 네이비 #000080) 기반으로 CSS 변수 미세 조정
- 섹션별 여백 확대 (padding 90~96px로 통일)
- 카드 border-radius 16px → 18px, padding 여유 확보
- 투명도/그라데이션 값 미세 조정으로 시각적 통일감 강화

### ✨ 애니메이션 효과 개선
- **Stagger 딜레이**: 같은 섹션 내 요소들이 순차적으로 등장 (0.08s 간격)
- **숫자 카운터**: `stats` 섹션에서 핵심 숫자가 0에서 목표값까지 easeOutCubic으로 증가
- **플로팅 CTA**: 스크롤 400px 이후 fade-up으로 자연스럽게 등장
- **헤더 전환**: 로고 크기 축소 + 배경 투명도 전환이 부드러운 transition으로 적용
- **fade-in 클래스 추가**: opacity만 변경하는 별도 애니메이션 타입

### 📖 가독성 개선
- line-height 전반적으로 1.75 → 1.8~1.95로 상향
- 섹션 타이틀 `letter-spacing: -.01em` 적용으로 가독성 향상
- FAQ 답변 영역 padding 확대 및 max-height 300px으로 여유 확보
- 후기 카드 내부 padding 확대, 텍스트 간 여백 개선
